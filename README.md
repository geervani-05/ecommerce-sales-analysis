# ecommerce-sales-analysis
E-Commerce Sales Analysis using Python and Power BI

## Project Overview
This project analyzes e-commerce sales data using Python and Power BI to uncover sales trends, customer behavior, and product performance.

## Tools & Technologies
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Power BI

## Objectives
- Analyze sales trends and revenue patterns
- Identify top-performing products
- Understand customer purchasing behavior
- Create interactive business dashboards

## Dataset
The project uses the Olist E-Commerce dataset. A compressed version of the dataset is included in this repository.

## Files Included
- E_Commerce_Sales_Analysis.ipynb → Python analysis notebook
- E-Commerce Sales Analysis Dashboard.pbix → Power BI dashboard
- E-Commerce Sales Analysis Dashboard.png → Dashboard screenshot
- merged_olist_dataset.zip → Dataset

## Key Insights
- Identified top-selling products and categories
- Analyzed revenue trends over time
- Created KPI dashboards for business decision-making
- Visualized sales performance using Power BI
